Jammin’ Gemstones

Entdecken Sie unterirdische Schätze – in Jammin’ Gemstones! Stimmen Sie 3 oder mehr benachbarte Edelsteine derselben Farbe überein, um Cluster zu bilden. Die Cluster verschwinden und die Punktzahl für jeden Edelstein erhöht sich, wenn neue Edelsteine von oben herabfallen. Gewinnen Sie einen Preis, wenn Sie genügend Edelsteine der gleichen Farbe sammeln. Lösen Sie den Edelstein-Ansturm-Bonus aus, indem Sie 4 Goldringe sammeln – Sie können sogar einen Sofortpreis gewinnen! Sammeln Sie 4 Silberringe und lösen Sie den Mischen-Bonus aus: Die Edelsteine können ihre Position auf dem Raster ändern, wo sie zufällig eingesammelt werden können, um neue Cluster zu bilden und Ihnen bessere Chancen auf einen Preis zu bescheren!

So wird gespielt

Ticketpreis

Zeigt den aktuellen Ticketpreis an.

Drücken Sie auf den linken Pfeil (–), um den Ticketpreis zu verringern.

Drücken Sie auf den rechten Pfeil (+), um den Ticketpreis zu erhöhen.

Drücken Sie auf KAUFEN, um den Ticketpreis vom Guthaben abzuziehen und das Spiel zu spielen.

Spielanleitung

Das Ziel bei Jammin’ Gemstones besteht darin, 3 oder mehr benachbarte Edelsteine derselben Farbe übereinzustimmen.

3 oder mehr benachbarte Edelsteine oder Ringe mit einem gleichfarbigen mittleren Juwel auf dem Raster erzeugen Cluster. Die Cluster werden entfernt und die Punktzahl für jede Edelsteinfarbe erhöht sich entsprechend. Sammeln Sie die erforderliche Anzahl an Edelsteinen, um einen Preis zu gewinnen. Für jede Edelsteinfarbe stehen mehrere Preise zur Verfügung.

Sammeln Sie 4 Goldringe, um den Edelstein-Ansturm-Bonus auszulösen und zusätzliche Edelsteine zu sammeln. Decken Sie ein farbenfrohes Juwelensymbol auf, um sofort einen Preis zu gewinnen.

Sammeln Sie 4 Silberringe, um den Mischen-Bonus auszulösen. Edelsteine können ihre Position auf dem Raster ändern, wo sie zufällig eingesammelt werden können oder neue Cluster bilden.

Das gewonnene Bonusspiel wird ausgelöst, wenn keine Cluster mehr auf dem Raster sind.

Verwerfen Sie die Gewinnnachricht und sehen Sie sich das Spielergebnis an, indem Sie auf die entsprechende Nachricht drücken.

Regeln

Missbrauch oder Fehlfunktionen annullieren alle Gewinne und Spiele.

Bonusgewinne werden zum Gesamtgewinn-Preis-Wert addiert.

Sofortgewinne werden zum Gesamtgewinn-Preis-Wert addiert.

Gewinne werden in der ausgewählten Währung angezeigt.

Spielerentscheidungen, die während des Spiels getroffen werden, beeinflussen das Ergebnis des Spiels nicht.
