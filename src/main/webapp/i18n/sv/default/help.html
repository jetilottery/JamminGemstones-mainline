Jammin’ Gemstones

Upptäck underjordiska skatter i Jammin’ Gemstones! Matcha 3 eller fler angränsande ädelstenar i samma färg för att skapa kluster. Klustren försvinner och poängen för varje ädelsten ökar i takt med att nya ädelstenar faller ned från ovan. Vinn ett pris när du samlat in tillräckligt många ädelstenar i samma färg. Aktivera Gem Rush-bonusen genom att samla in 4 guldringar – du kan till och med vinna ett pris direkt! Samla in 4 silverringar och aktivera Shuffle-bonusen: ädelstenarna kan skifta position på rutnätet, där de samlas in slumpmässigt för att bilda nya kluster och ger ännu större chanser att vinna ett pris!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (–) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och börja spela.

Så här spelar du

Målet i Jammin’ Gemstones är att matcha 3 eller fler angränsande ädelstenar i samma färg.

3 eller fler angränsande ädelstenar, eller ringar med samma ädelstensfärg, på rutnätet skapar kluster. Klustren tas bort och poängen för varje ädelstensfärg ökar i enlighet därmed. Samla det antal ädelstenar som krävs för att vinna ett pris. Det finns flera priser tillgängliga för varje ädelstensfärg.

Samla in 4 guldringar för att aktivera Gem Rush-bonusen och samla in ytterligare ädelstenar. Avslöja en färgglad ädelstenssymbol för att vinna ett pris direkt.

Samla in 4 silverringar för att aktivera Shuffle-bonusen. Ädelstenar kan skifta position på rutnätet, där de samlas in slumpmässigt eller bildar nya kluster.

Ett bonusspel aktiveras när inga kluster finns kvar på rutnätet.

Avfärda vinstmeddelandet och se spelets utfall genom att trycka på meddelandet.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Bonusvinster läggs till det totala prisvärdet.

Direktvinster läggs till det totala prisvärdet.

Vinster visas i vald valuta.

Val som görs av spelare under spelets gång påverkar inte spelets utfall.
